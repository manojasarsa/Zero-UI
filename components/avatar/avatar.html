Avatar

The Avatar component is used to represent a user profile pictures on profile information page, on navigation bar, in blogs grid items. They usually found in circular shapes.

Profile Avatar

Profile Avatars can be created using the class 'avatar_image', inside the img tag.

Sizes

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size : avatar-lg, avatar-md, avatar-sm, avatar-xs (e.g. class="avatar avatar-md")
