Simplified Grid

CSS Grid is a two-dimensional grid system used to work on the layout of UI elements and segments of a webpage. Grid System creates complex responsive web design grid layouts more easily and consistently across browsers. It helps align page elements based on sequenced columns and rows.

(1x2) Grid :Two Items In A Grid

We have (1x2) grid. To use this you have to add grid_two_columns in parent container and child items should contain class grid_item.

(1x3) Grid :Two Items In A Grid

We have (1x3) grid. To use this you have to add grid_three_columns in parent container and child items should contain class grid_item.
