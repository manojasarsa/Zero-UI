Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Standard Alerts

The alert offers four severity levels that set a distinctive icon and color. These are : success alert, error alert, warning alert and info alert.

task_alt This is a success alert - check it out!
error_outline This is a error alert - check it out!
report_problem This is a warning alert - check it out!
info This is a info alert - check it out!
